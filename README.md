# 訂餐備註分析（小狀元）

這是用於訂餐資料整理、標籤分析、交叉表統計與 A4 列印輸出的工具。

## 文件入口（請先看這裡）

- 架構與目前功能：`docs/ARCHITECTURE.md`
- 打包、換機與懶人包流程：`docs/BUILD_AND_RELEASE.md`
- 抓取網頁 profile 規格：`docs/WEB_FETCH_PROFILE.md`
- 分析欄位設定：`docs/ANALYZE_FIELDS.md`

## 使用提醒

- 程式設定與資料檔（JSON）會寫在程式所在資料夾（EXE 模式）。
- 換電腦時，請連同設定 JSON 一起搬移。
- 不熟悉電腦操作者請使用「懶人包」版本（含一鍵啟動/還原設定）。
